NAME:
   fritzbox-graphite - Sends Fritz!Box Traffic Data to Graphite

USAGE:
   fritzbox-graphite [global options] command [command options] [arguments...]

VERSION:
   0.0.0

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --router, -r "fritz.box"      Fritz!Box Hostname or IP address
   --carbon-host, --ch "localhost"  Carbon Hostname or IP address
   --carbon-port, --cp "2003"    Carbon Port
   --interval, -i "1m0s"      Interval
   --prefix "metrics.routers.7170"  Prefix
   --help, -h           show help
   --version, -v        print the version
